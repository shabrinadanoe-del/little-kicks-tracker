# Little Kicks Tracker

Simple baby movement tracker with an earth-tone UI.

## Features
- One-tap logging whenever the baby moves
- Daily movement totals
- Peak active hours summary
- Per-hour activity chart
- Cross-date history view
- Notes / symptoms per date for OB-GYN visits
- Data stored in browser localStorage

## Run locally
```bash
npm install
npm run dev
```

Open `http://localhost:3000`

## Deploy to Vercel
1. Upload this project to GitHub, or drag-and-drop it into Vercel.
2. Framework preset: Next.js
3. Build command: `next build`
4. Output: default

## Notes
This app is for personal tracking only and not a medical diagnostic tool.
